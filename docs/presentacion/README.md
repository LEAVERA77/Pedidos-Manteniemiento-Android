# Presentaciones GestorNova

Documentos para reuniones comerciales y presentación del producto.

| Archivo | Formato | Audiencia |
|---------|---------|-----------|
| `GestorNova-Cooperativa-Electrica.md` | **Markdown (recomendado)** | Cooperativas y distribuidoras eléctricas |
| `GestorNova-Municipios.md` | **Markdown (recomendado)** | Municipalidades |
| `GestorNova-Cooperativa-Electrica.docx` | Word (opcional) | Misma audiencia |
| `GestorNova-Municipios.docx` | Word (opcional) | Municipalidades |

Abrí los `.md` en Cursor, VS Code o cualquier visor Markdown; también se pueden exportar a PDF desde el IDE.

## Regenerar Word (opcional)

```powershell
powershell -ExecutionPolicy Bypass -File docs/scripts/generar-presentaciones-gestornova-word.ps1
```

Los `.md` se editan directamente en `docs/presentacion/`.

Documentación técnica de masivos WhatsApp: `api/docs/WHAPI_BROADCAST_COMPLIANCE.md`
